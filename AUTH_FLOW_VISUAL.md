## Migration Path

### For Existing Users (Production)
**No action required.** Users will automatically benefit from:
- Session persistence
- Better error messages
- More reliable authentication

### For Developers

**Before:**
```bash
# Dev mode with auto-mock
npm run dev
# Mock auth automatically active
```

**After:**
```bash
# Default dev mode (no mock)
npm run dev

# OR enable mock auth when needed
VITE_ENABLE_MOCK_AUTH=true npm run dev
```

Add to `.env.local` for persistent mock auth:
```env
VITE_ENABLE_MOCK_AUTH=true
```
